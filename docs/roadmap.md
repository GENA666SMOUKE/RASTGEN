# RASTGEN roadmap

## v0.1.0

- Initial CLI
- Repository audit
- Issue triage
- PR summary
- Changelog draft
- Risk check
- GitHub Action metadata
- CI workflow
- Documentation and examples

## v0.1.1

- Robust GitHub Action input handling for dashed and underscore input names.
- Scoring-based issue triage to reduce false positives.
- Conventional Commit-first changelog categorization.
- Risk analysis that distinguishes free text, diffs, and file lists.
- Clearer audit-score wording and limitations.
- Regression tests for the fixes above.

## v0.2.0

- JSON output mode
- Markdown report output
- Config file support
- More repository audit checks
- Structured risk scoring output
- Optional repository-specific triage rules

## v0.3.0

- GitHub API integration
- Automatic PR comments
- Automatic issue label suggestions
- Release note drafts from merged pull requests
- Maintainer workflow reports

## Future

- Optional AI-assisted issue triage
- Optional AI-assisted PR review summaries
- Security-sensitive diff analysis
- Documentation freshness checks
- Dependency risk summaries
- Maintainer dashboard
- Add Markdown report output for CI summaries.
