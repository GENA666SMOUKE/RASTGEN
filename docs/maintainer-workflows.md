# Maintainer workflows

RASTGEN is designed around common maintainer workflows.

## New repository setup

Use `audit` to check whether a repository is ready for external contributions.

## Incoming issue triage

Use `issue-triage` to suggest a first label before manual maintainer review.

## Pull request review

Use `pr-summary` to quickly understand what areas changed.

Use `risk-check` to detect high-risk terms before merging.

## Release preparation

Use `changelog` to turn a raw list of commits or merged changes into a structured changelog draft.

## Future AI-assisted workflow

Future versions may optionally use LLMs to generate higher-quality PR summaries, changelog drafts, issue triage suggestions, and documentation checks.
